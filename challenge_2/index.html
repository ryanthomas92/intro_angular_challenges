Challenge #2: Creating Controllers

  1. Copy paste the follwing script tag into the <head> tag:
    <script src="https://cdnjs.cloudflare.com/ajax/libs/angular.js/1.4.9/angular.js"></script>
  2. Add the attribute "ng-app" to the html tag, like so:
    <html ng-app="appTwo">
    (Note that we're adding an app name this time!)
  3. You should see a nasty error message in your console now:
    Failed to instantiate module appTwo due to:... Module 'appTwo' is not available! You either misspelled the module name or forgot to load it. If registering a module ensure that you specify the dependencies as the second argument.
    - Let's add the following code to make it go away (js/app.js):
    + To make that error go away, we need to use angular to create a module called appTwo. We'll start out the module with no extra dependencies. Here's the code to add (js/app.js):
    var app = angular.module("appTwo", []);
  4. This box will be checked if appTwo is working: <-- STOP! Don't proceed until this is checked! (Note: you'll still see an error in the browser console)
  5. The new error is looking at the WelcomeController section later in the HTML, but it can't find a WelcomeController function in the JavaScript! Let's create our first controller and add it to the appTwo module (js/app.js):
    var app = angular.module("appTwo", []);
app.controller("WelcomeController", WelcomeController);

function WelcomeController(){
  this.full_name = "YOUR NAME"; // change me
}
  6. You should see your name displayed: {{ your.full_name }}
  7. Before doing the exercises, can you find where the below WelcomeController code is referenced in the html?
    (This is the glue that links your controller to your view!) 
    <div ng-controller="WelcomeController as welcomeCtrl">
  <!-- the magic happens in here -->
<div>
  8. OK! Angular is up and running and you have a controller. Let's give the following exercises a shot!
